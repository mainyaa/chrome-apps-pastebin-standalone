Pastebin for Chrome App
====

Build
----

 * `npm install`
 * `grunt build`

Develop
----

 * `npm install`
 * `grunt debug`

Install
----

 * Open Chrome

 * Go to the Extensions screen

 * Expand "Developer Mode"

 * Click "Load unpackaged extension..."
 * Choose `app` or `dist` directory of this repo
